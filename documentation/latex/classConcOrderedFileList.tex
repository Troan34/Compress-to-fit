\subsubsection*{template$<$\doxymbox{\hyperlink{conceptSerializableToDisk}{Serializable\+To\+Disk}} T$>$\newline
class Conc\+Ordered\+File\+List$<$ T $>$}
First sequence in first sequence out, i.\+e. the data is written in the order given to \doxylink{classConcOrderedFileList_a2c53a81b8da07b0b7e0502ad83d98e0d}{insert()}. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Must implement write\+\_\+to(ofstream\&) or be trivially copyable\\
\hline
\end{DoxyTemplParams}
Implemented as doubly linked list. A separate write thread is spawned.

\begin{DoxyNote}{Note}
Because the list will own the elements, consider using a memory pool (e.\+g. pmr) as to not reallocate memory. 
\end{DoxyNote}
